This directory contains the configuration files for this program.
program.ini contains the default values for Assetto Corsa configurations that don't change based on the car data (version, model, etc.)

#config stuff
testing.ini contains the values used to test the program. The correct values for the system that is running the tests must be entered, one per line as shown below.
